# Scripted Pass Plan: Retire Stale `gunner/` Prefix Wikilinks

## Background

The vault was restructured at some point from a flat `gunner/` top-level folder to
the current per-owner layout (`gunnerteam/`, `tyler/gunnerteam/`,
`tyler/masterdb/`, etc.). [[meta/lint-report-2026-07-02]] found **301 wikilink
occurrences across 113 files** still using the retired `gunner/` prefix (down
from 333/118 before that lint's ambiguous-collision fix — see below).

These links are **not currently broken**: Obsidian's resolver falls back to
matching by bare filename when a full path doesn't resolve, and every one of
these 31 distinct targets currently has a unique filename vault-wide. But this
is fragile — the first time a new file is added anywhere in the vault sharing
one of these 31 basenames, the fallback becomes ambiguous (silently
non-deterministic in Obsidian) with zero warning, exactly like the
`gunner/system-security-plan` case just fixed (2 files shared that basename;
37 links across 22 files were resolving into a real collision).

## What Was Already Fixed (2026-07-02, same session)

The one genuinely ambiguous cluster — `[[gunner/system-security-plan]]` (basename
collision between `gunnerteam/system-security-plan.md` and
`tyler/summaries/system-security-plan.md`) — was fixed immediately as part of
this lint cycle: 37 wikilink occurrences across 22 files repointed to
`[[gunnerteam/system-security-plan]]`, the canonical page linked from
`wiki/index.md`. One adjacent plain-text historical reference
(`wiki/gunner/system-security-plan.md`, in a session note documenting a past
bug) was deliberately left untouched — it is not a wikilink and does not
navigate.

**This plan covers everything else**: the remaining 301 occurrences, all
currently non-ambiguous (unique basename today), across 113 files.

## The Script

Idempotent, dry-run by default, only touches `[[gunner/<exact-target>]]` forms
(bare, piped, or with a `#anchor`) for the 31 mapped targets — nothing else.
Skips `wiki/log.md` and any `wiki/meta/lint-report-*.md` / `wiki/meta/lint/*`
file (those are historical records of past lint findings, not live content —
same exclusion rule the lint skill itself uses).

```python
#!/usr/bin/env python3
"""Retire stale gunner/ prefix wikilinks -> canonical paths.
Usage:
    python3 fix_gunner_prefix_links.py           # dry run, prints diff summary only
    python3 fix_gunner_prefix_links.py --apply    # writes changes
"""
import re
import sys
from pathlib import Path

VAULT = Path(__file__).resolve().parent  # run from vault root, or edit this
WIKI = VAULT / "wiki"

MAPPING = {
    "gunner/environment": "gunnerteam/environment",
    "gunner/app-inventory": "gunnerteam/app-inventory",
    "gunner/gunnerteam-api-aws-migration": "tyler/gunnerteam/gunnerteam-api-aws-migration",
    "gunner/gunner-forms-app": "gunnerteam/gunner-forms-app",
    "gunner/hubspot-leads-project": "gunnerteam/hubspot-leads-project",
    "gunner/aws-environment": "gunnerteam/aws-environment",
    "gunner/lead-assignment-automation": "gunnerteam/lead-assignment-automation",
    "gunner/software-suite": "gunnerteam/software-suite",
    "gunner/hubspot-sales-pipeline": "gunnerteam/hubspot-sales-pipeline",
    "gunner/gunnerteam-project-structure": "tyler/gunnerteam/gunnerteam-project-structure",
    "gunner/masterdb-architecture": "tyler/masterdb/masterdb-architecture",
    "gunner/dialpad-hubspot-integration": "gunnerteam/dialpad-hubspot-integration",
    "gunner/masterdb-developer-handoff": "tyler/masterdb/masterdb-developer-handoff",
    "gunner/federal-market": "gunnerteam/federal-market",
    "gunner/hubspot-salesperson-sop": "gunnerteam/hubspot-salesperson-sop",
    "gunner/brand-colors": "gunnerteam/brand-colors",
    "gunner/claude-session-onboarding": "gunnerteam/claude-session-onboarding",
    "gunner/gunner-assistant": "tyler/gunner-assistant/gunner-assistant",
    "gunner/chrome-policy": "gunnerteam/chrome-policy",
    "gunner/claude-team-setup": "gunnerteam/claude-team-setup",
    "gunner/completed-projects": "gunnerteam/completed-projects",
    "gunner/secure-coding-guide": "gunnerteam/secure-coding-guide",
    "gunner/hubspot-workflow-designs": "gunnerteam/hubspot-workflow-designs",
    "gunner/it-decision-log": "gunnerteam/it-decision-log",
    "gunner/subportal-cc-prompt-01-scaffold": "tyler/gunnerteam/subportal-cc-prompt-01-scaffold",
    "gunner/subportal-cc-prompt-02-frontend": "tyler/gunnerteam/subportal-cc-prompt-02-frontend",
    "gunner/subportal-cognito-auth": "tyler/gunnerteam/subportal-cognito-auth",
    "gunner/gunnerteam-performance-standards": "tyler/gunnerteam/gunnerteam-performance-standards",
    "gunner/departmental-comms": "gunnerteam/departmental-comms",
    "gunner/gunner-forms-privacy-policy": "gunnerteam/gunner-forms-privacy-policy",
    "gunner/tls-cutover-2026-05-14": "gunnerteam/tls-cutover-2026-05-14",
}

SKIP_FILES = {"wiki/log.md"}
def is_skipped(rel: str) -> bool:
    if rel in SKIP_FILES:
        return True
    base = Path(rel).name
    if base.startswith("lint-report-"):
        return True
    if "meta/lint/" in rel:
        return True
    return False

# Matches [[gunner/<target>]], [[gunner/<target>|alias]], [[gunner/<target>#anchor]]
LINK_RE = re.compile(r"\[\[(" + "|".join(re.escape(k) for k in MAPPING) + r")([#|][^\]]*)?\]\]")

def fix_text(text: str) -> tuple[str, int]:
    count = 0
    def repl(m):
        nonlocal count
        count += 1
        target, suffix = m.group(1), m.group(2) or ""
        return f"[[{MAPPING[target]}{suffix}]]"
    new_text = LINK_RE.sub(repl, text)
    return new_text, count

def main():
    apply = "--apply" in sys.argv
    total_files = 0
    total_occurrences = 0
    for f in sorted(WIKI.rglob("*.md")):
        rel = f.relative_to(VAULT).as_posix()
        if is_skipped(rel):
            continue
        text = f.read_text(encoding="utf-8")
        new_text, count = fix_text(text)
        if count:
            total_files += 1
            total_occurrences += count
            print(f"{'FIXED' if apply else 'WOULD FIX'}: {rel} ({count} occurrence{'s' if count != 1 else ''})")
            if apply:
                f.write_text(new_text, encoding="utf-8")
    print()
    print(f"{'Applied' if apply else 'Would apply'}: {total_occurrences} substitutions across {total_files} files.")
    if not apply:
        print("Re-run with --apply to write changes.")

if __name__ == "__main__":
    main()
```

1. `cd ~/Documents/Obsidian/gunner-brain && git pull --rebase origin main`
2. Save the script above as e.g. `/tmp/fix_gunner_prefix_links.py`.
3. Dry run first: `python3 /tmp/fix_gunner_prefix_links.py` (run with cwd =
   vault root) — confirm the file count and occurrence count match this plan
   (113 files, 298 occurrences) before writing anything. A mismatch means the
   vault changed since this plan was written — re-derive the mapping table
   before proceeding, don't blindly apply a stale plan.
4. Apply: `python3 /tmp/fix_gunner_prefix_links.py --apply`
5. Spot-check a handful of diffs (`git diff -- <file>`) across different

## Why This Is a Separate Pass, Not Part of the 2026-07-02 Lint

113 files is too large a diff to review confidently inside a single lint
session alongside everything else that pass covers — the risk isn't the
substitution logic (it's a precise, tested exact-match replace), it's
verifying none of it silently changes meaning in files with unusual formatting
outside the `[[...]]` bracket syntax. Running it standalone, with its own
commit and its own before/after grep-verification, keeps a bad substitution
easy to `git revert` in isolation rather than tangled into an otherwise-clean
lint commit.
