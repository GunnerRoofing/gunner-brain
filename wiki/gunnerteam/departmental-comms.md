# Departmental Communications Guide

Which tool to use for what, by department and use case. Source of truth for communication channel decisions at Gunner.

## Tool-to-Use-Case Map

| Tool | Primary Purpose | Use Cases |
|------|----------------|-----------|
| **Google Chat** | All company communication | All Gunner announcements, internal project communication, IT request management, marketing, team communication |
| **Gmail** | Formal/external email | Google Drive link sharing, meeting invitations, reimbursement requests, PTO requests, corporate requests |
| **Dialpad** | Customer communication | Peer-to-peer phone/SMS, customer-facing calls |
| **Monday.com** | Operations project management | IT development, IT request tracking, project management |
| **HubSpot** | Sales CRM | Sales pipeline, lead management |
| **CompanyCam** | Field documentation | Project photos, daily logs |
| **Quote Portal** | Sales quoting | Quote customization and management |
| **Forms (Monday.com)** | Internal requests | IT Request Form, Service Request Form, Referral Request Form, PTO Request Form, Crew Payouts |

## Key Channel Rules

- **Customer-facing communication** → Dialpad (not Google Chat, not Gmail for day-to-day)
- **Internal project updates** → Google Chat
- **IT requests** → Monday.com IT board (via GunnerForms IT Request form or Monday directly)
- **Formal documents / file sharing** → Gmail
- **Sales activity** → HubSpot (not Monday, not Google Chat)
